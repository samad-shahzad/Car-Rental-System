# Car Rental System

A console-based Car Rental System developed in C++ as a Programming Fundamentals semester project.

## Features

* Add and manage cars
* Rent cars to customers
* Return rented cars
* Display available cars
* Store and manage customer information

## Technologies

* C++
* Programming Fundamentals
* Console Application

## Purpose

This project was developed as a semester project to practice programming fundamentals, problem-solving, and application development using C++.
